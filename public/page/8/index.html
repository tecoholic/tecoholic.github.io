Python Tip #9 - sorting

Sorting is simplified in Python with sorted(). You can even sort with complex rules. >>> strings = ['alice', 'bob', 'donald', 'cathy'] >>> sorted(strings) ['alice', 'bob', 'cathy', 'donald'] >>> sorted(strings, key=len) ['bob', 'alice', 'cathy', 'donald'] >>> def secondchar(word): ... return word[1] >>> sorted(strings, key=secondchar) ['cathy', 'alice', 'bob', 'donald']
